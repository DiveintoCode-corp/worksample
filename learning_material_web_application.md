# 推奨学習教材

英語のサービスは、ブラウザの翻訳機能を使い、翻訳すれば容易に学習ができます。
なお、弊社に著作権が帰属していないサービスがあります。 `それらの教材に関する質疑応答はできません。` あらかじめご了承ください。
